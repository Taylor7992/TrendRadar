这个是我总结的后续如果需要将项目部署到本地的一般过程
1️⃣ GitHub → Download ZIP  
2️⃣ 解压到：D:\ 
3️⃣ 打开 Clash（系统代理开启）  
4️⃣ 启动 Docker Desktop  
5️⃣ docker build 构建镜像  （需要确定端口在对应的flash中找） 
6️⃣ docker run 挂载 config 运行  
7️⃣ 修改 config.yaml 控制程序行为

针对于这个项目
在此路径下D:\tools\TrendRadar\config的frequency_words我们可以进行一个关键词的选择（简单来说就是你感兴趣的内容）
而在D:\tools\TrendRadar\config的config.yaml中我们可以选择发送信息的方式，我在这里选择的是gmail，还知道了关于16位的授权码（这是一个不同于登录密码的东西）
