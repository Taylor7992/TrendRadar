1️⃣ GitHub → Download ZIP  
2️⃣ 解压到：D:\ 
3️⃣ 打开 Clash（系统代理开启）  
4️⃣ 启动 Docker Desktop  
5️⃣ docker build 构建镜像  （需要确定端口在对应的flash中找） 
6️⃣ docker run 挂载 config 运行  
7️⃣ 修改 config.yaml 控制程序行为

如何定期清理 Docker（强烈推荐 1 个月一次）
🧹 删除未使用的容器/镜像/volume（安全清理）
docker system prune
🧹 删除所有未使用的镜像（更彻底）
docker system prune -a
🧹 删除未使用的 volumes（注意：会删除未挂载卷）
docker volume prune
